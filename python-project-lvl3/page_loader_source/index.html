PHP-программист

PHP — серверный язык программирования. Создан специально для бэкенд-разработки. PHP проще для новичков, чем большинство других серверных языков. Slack, Wikipedia, ВКонтакте используют PHP.

Java-программист

Java — универсальный язык программирования. Работает на любой архитектуре и может встраиваться в веб-приложения. Широко используется в разработке десктоп-, веб- и мобильных приложений, в банковской, научной и промышленной областях.

Хотите развиваться

На Хекслете есть продвинутые курсы для действующих программистов. Они помогают расширить кругозор, получить фундаментальные знания, познакомиться с правильными инженерными практиками, освоить новые инструменты.

Как проходит обучение

Вы выбираете профессию, мы подбираем для вас индивидуального наставника. Это действующий программист с опытом преподавания. Обучение программированию в рамках профессии объединяется в четыре модуля. Каждый модуль состоит из тематических курсов и практического проекта.

Проходите теорию, читаете дополнительную литературу, обсуждаете с наставником сложные вопросы

После каждого теоретического урока проходите тестирование. Это нужно, чтобы закрепить материал и проверить себя

Решаете практическое задание. Упражнения учат студентов применять теоретические знания для решения реальных задач

В конце каждого модуля вы реализуете практический проект

На проекте необходимо самостоятельно настроить окружение и написать полноценное работающее приложение. Затем под руководством наставника и менторов студент выполняет рефакторинг: исправляет ошибки, оптимизирует код приложения.

Проекты — главная точка роста для студентов Хекслета. Работа над приложением позволяет курсанту «пощупать руками» реальную разработку, прокачаться, подготовиться к встрече с настоящими задачами и продакшен-кодом.

Обучение на Хекслете: результаты и отзывы

Работодатели ценят кандидатов, которые прошли обучение в нашей школе программирования. Благодаря этому студенты Хекслета быстро находят работу. Подробности можно найти в историях успеха выпускников.

Хекслет привлёк комплексностью материала курсов. С одной стороны тебе даётся именно тот минимум, который уже позволяет решать те или иные задачи, а с другой раскрываются вопросы, которые изначально не кажутся важными, но они необходимы для дальнейшего развития. Это то, чего невозможно добиться при самостоятельном изучении по книгам, когда ты просто не знаешь, что именно надо изучить в огромном массиве доступных знаний.
